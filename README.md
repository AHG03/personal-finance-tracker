# Personal Finance Tracker (Python)

A simple command-line personal finance tracker written in Python.
Users can add income and expense transactions, view them, and persist data using JSON.

## Features
- Add income and expense transactions
- View all transactions
- Persistent storage using JSON
- Object-oriented design

## Tech Stack
- Python 3
- JSON for persistence

## What I learned
- Designing data models with classes
- Separating business logic from user interaction
- Converting between runtime objects and serialized data
- Managing application state

## Possible improvements
- Input validation
- SQLite database support
- Web interface using Flask
